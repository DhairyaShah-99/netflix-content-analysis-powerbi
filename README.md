# Netflix Content Analysis (Power BI)

This project explores the Netflix dataset using Power BI to analyze trends in movies and TV shows.

## Key Insights
- Distribution of movies vs TV shows
- Content availability across different countries
- Popular genres on Netflix
- Release year trends of Netflix content

## Tools Used
Power BI  
Data Visualization  
Data Analysis

## Dataset
Netflix Movies and TV Shows dataset.
